## 기도문 톤 분류

각 기도문은 세 가지 톤으로 구성됩니다:

| 톤 | 설명 | 특징 |
|----|------|------|
| 은혜롭고 감동적인 | 격식 있고 유려한 문체 | 하나님의 속성 풍성히 묘사, 시적 표현 |
| 위로와 회복 | 따뜻하고 공감적인 문체 | 위로 어휘, 희망적 결말 강조 |
| 친근한 | 쉽고 담백한 문체 | 일상적 표현, 솔직한 고백 |

## 사용 가이드

### 기도문 선택 기준
1. **상황**: 어떤 모임/예배인지 확인
2. **톤**: 그날 분위기에 맞는 톤 선택
3. **시간**: 분량 확인 (대표기도 5~8분 / 기타 1~2분)

### 기도문 활용 팁
- 그대로 읽어도 좋지만, 그날의 상황에 맞게 일부 수정하면 더 좋습니다
- [핵심 패턴](./patterns/core-patterns.md)을 참고하여 자유롭게 조합할 수 있습니다
- 기도 전 잠시 묵상하며 마음을 가다듬고 시작하세요
