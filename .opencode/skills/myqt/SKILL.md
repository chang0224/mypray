# MyQT - 매일성경 묵상 노트 추출

두란노 생명의삶(매일성경) 웹사이트에서 오늘의 QT 콘텐츠를 추출합니다.

## Fetching

사이트는 **EUC-KR** 인코딩을 사용합니다. 반드시 `iconv`로 변환해야 합니다.

```bash
curl -s "https://www.duranno.com/qt/view/bible.asp?qtDate=${DATE}&d=k" | iconv -f euc-kr -t utf-8
```

- `${DATE}` 형식: `YYYY-MM-DD` (예: `2026-06-15`)
- 날짜가 지정되지 않으면 오늘 날짜 사용 (`date +%Y-%m-%d`)
- `&d=k`는 개역개정 역본 (기본값)

## Parsing (HTML에서 추출할 섹션)

### 1. 본문 정보 (제목, 범위)

```html
<h1><span>고린도전서  15 : 29~34</span><em>고난을 인내하고 죄를 이기는 부활 소망</em></h1>
```

- `<h1>` 안의 `<span>` = 성경 범위
- `<h1>` 안의 `<em>` = 묵상 제목

### 2. 찬송 (오늘의 찬송)

```html
<div class="song box">
  <p class="title">오늘의 찬송</p>
  <p>(새 149 주 달려 죽은 십자가)<br><br>가사 내용...</p>
</div>
```

### 3. 본문 말씀

```html
<div class="bible">
  <p class="title">소제목</p>
  <table><tr><th>절번호</th><td>본문 내용</td></tr></table>
  ...
</div>
```

- `<p class="title">` = 단락 소제목
- `<th>` = 절 번호, `<td>` = 본문 텍스트

### 4. 오늘의 기도

```html
<div class="helper box matop">
  <p class="title">오늘의 기도</p>
  <p>기도 내용...</p>
</div>
```

## File Storage

추출한 QT 콘텐츠는 프로젝트 루트의 `myQT/` 디렉토리에 저장합니다.

### 디렉토리 구조

```
myQT/
├── index.md          ← 전체 인덱스 (자동 갱신)
├── 2026/
│   ├── 2026-06-15.md
│   ├── 2026-06-16.md
│   └── ...
├── 2025/
│   └── ...
└── ...
```

### 저장 규칙

1. **파일명**: `YYYY-MM-DD.md` (날짜 기반)
2. **연도 디렉토리**: `myQT/YYYY/` — 없으면 생성
3. **파일 경로**: `myQT/YYYY/YYYY-MM-DD.md`
4. **파일 내용**: Output Format 섹션의 마크다운 형식 그대로 저장

### 인덱스 파일 (`myQT/index.md`)

파일 저장 후 반드시 `myQT/index.md`를 갱신합니다.

```markdown
# 매일성경 묵상 노트

## 2026

| 날짜 | 본문 | 제목 |
|------|------|------|
| [2026-06-15](2026/2026-06-15.md) | 고린도전서 15:29~34 | 고난을 인내하고 죄를 이기는 부활 소망 |
| [2026-06-14](2026/2026-06-14.md) | 고린도전서 15:20~28 | ... |

## 2025

| 날짜 | 본문 | 제목 |
|------|------|------|
| ... | ... | ... |
```

- 연도별 섹션(`## YYYY`)으로 구분
- 각 섹션 내에서 날짜 내림차순 (최신이 위)
- 인덱스에 이미 존재하는 항목은 중복 추가하지 않음
- 새 연도가 추가되면 연도 섹션도 내림차순 정렬

## Notes

- HTML 태그 제거 후 순수 텍스트만 추출
- `<br>` 태그는 줄바꿈 또는 `/`로 처리
- 찬송 가사의 `/`는 절 구분
- 날짜의 요일은 사이트에서 제공하는 값 사용 (예: `2026.06.15 (월)`)
